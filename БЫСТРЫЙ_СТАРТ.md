# Быстрый старт

## Минимальные шаги для запуска

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Получение токена
1. Найдите @BotFather в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям
4. Скопируйте токен

### 3. Настройка токена
```bash
export BOT_TOKEN="ваш_токен_здесь"
```

Или отредактируйте `config.py`:
```python
BOT_TOKEN = os.getenv("BOT_TOKEN", "ваш_токен_здесь")
```

### 4. Настройка администратора (опционально)
1. Узнайте свой ID через @userinfobot
2. Добавьте ID в `config.py` в список `ADMIN_IDS`

### 5. Запуск
```bash
python bot.py
```

### 6. Добавление тестовых данных (опционально)
В другом терминале:
```bash
python add_sample_data.py
```

### 7. Проверка
1. Найдите вашего бота в Telegram
2. Отправьте `/start`

## Основные команды

- `/start` - Приветствие
- `/help` - Справка
- `/search Иванов` - Поиск
- `/list` - Список всех
- `/status протоиерей` - Фильтр по статусу
